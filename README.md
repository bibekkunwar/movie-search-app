Name: Movie Search app.
Description: A movie search app using TMBD api that fetches movie list from the search keyword.
Features: 
1. Rows of movie list. Each row with 10 movies.
2. Pop up modal to show movie Overview.
3. Error message on failed request.

Tech Stacks: JSX, html, tailwindcss

Live demo: https://bibekkunwar.github.io/movie-search-app/
